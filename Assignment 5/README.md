## File Description

- `level5.sh`: Automates server interaction for encrypting plaintexts.
- `find_mat.ipynb`: Determines the elements of the lower-triangular matrix A.
- `find_e_diag.ipynb`: Finds the exponent vector E and diagonal of matrix A.
- `mat_inv.ipynb`: Calculates the inverse of the recovered matrix A.
- `decoding.ipynb`: Decodes the final password using all recovered components.
- `cryptanalysis.py`: Holds primary functions for the cryptanalytic attack.
- `supporting_functions.py`: Contains helper functions for data conversion and utilities.
- `constants.py`: Stores fixed values for cryptographic operations.
